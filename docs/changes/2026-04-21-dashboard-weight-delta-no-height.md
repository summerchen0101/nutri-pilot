# Dashboard 體重卡：對比上一筆、色彩、隱藏身高

**日期**：2026-04-21  
**影響規格**：[`docs/06-pages.md`](../06-pages.md)（`/dashboard` 體重卡描述）  
**類型**：修改

## 原規格

- 體重卡顯示今日或最近體重與 BMI 計算值（[`docs/06-pages.md`](../06-pages.md) 表格列「體重卡」）。

## 實際做法

- 首頁體重卡副行改為：若有至少兩筆 `vital_logs` 且兩筆皆有體重，顯示與**上一筆紀錄**的體重差（一位小數、帶正負號與 `kg`）；增加為輔色橘 `#EF9F27`，減少為主色綠（`text-primary`），零為 `text-muted-foreground`。
- 同一行以 ` · ` 接 BMI（輔助色），**不再**顯示身高。
- 僅一筆或無法相減時不顯示對比數字；若仍有 BMI 則僅顯示 `BMI …`。

## 原因

產品需求：強調短期體重變化並與設計系統色彩一致，首屏不暴露身高。

## 後續

可將 `docs/06-pages.md` 中 `/dashboard` 體重卡列更新為上述行為。
