# Dashboard：連續達成空狀態改為個人化問候

**日期**：2026-04-21
**影響規格**：docs/06-pages.md
**類型**：修改

## 原規格

Dashboard Header 主要描述「問候語 + 今日日期 + 連續打卡 badge」，未明確定義 streak 為 0 的顯示內容。

## 實際做法

當 `streakDays >= 1` 時，維持顯示「連續 N 天達成！」badge。  
當 `streakDays < 1` 時，左側改顯示「含使用者名字的問候 / 今日建議」文案，不再留白。

## 原因

避免首頁左上區塊在無 streak 時視覺留空，並提升首頁的個人化與引導性。

## 後續

可視需要將問候語句庫抽到 i18n / 文案配置，方便後續 A/B 測試與語氣統一。
