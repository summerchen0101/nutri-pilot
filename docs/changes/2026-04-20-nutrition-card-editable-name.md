# NutritionResultCard 可編輯食物名稱

**日期**：2026-04-20  
**影響規格**：docs/09-ui-design.md（/log 手動與拍照營養卡）  
**類型**：修改

## 原規格

- 營養結果卡上的食物名稱僅能顯示，不可在寫入前修改。

## 實際做法

- 名稱改為可點擊切換的 inline 文字輸入；失焦或 Enter 結束編輯，若內容空白則還原為 AI 基準名稱（`originalResult.name`）。
- 鉛筆圖示常駐、淡色（`opacity-60`）以符合觸控裝置（無 hover 依賴）。
- 加入紀錄時以 `displayName`（去空白後、空則用基準名）寫入；變更份量比例時保留目前顯示名稱。

## 原因

讓使用者在寫入前可微調顯示用名稱，與實務記錄習慣一致。

## 後續

無。
