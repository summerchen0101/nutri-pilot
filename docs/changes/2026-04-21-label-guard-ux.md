# 標籤守衛：警示說明浮層、風險標籤配色、過敏原僅列陽性

**日期**：2026-04-21  
**影響規格**：docs/06-pages.md  
**類型**：修改

## 原規格

`/guard` 結果區呈現安全分數、警示關鍵字、族群建議、風險分級與 **14 類過敏矩陣**（文件中未區分「全列表」或「僅陽性」，實際 UI 為 14 類一律列出）。

## 實際做法

1. **警示關鍵字、成分風險列、過敏原陽性列**可點開底部說明（`BottomSheetShell`）；常見警示詞與 14 類過敏原另有繁中說明查表（`src/lib/food/label-guard-lookups.ts`），查無表列時使用免責／一般性說明。
2. **風險分級標籤**：低風險為主色綠系 pill，其餘 tier 為橘／琥珀系背景與字色以提高對比。
3. **過敏原區塊**：僅列出 `detected === true` 之類別；若皆未偵測則顯示一句輔助說明。

## 原因

產品需求：可讀性、可操作之說明入口，並減少「未標示／未見」對使用者的噪音。

## 後續

若有需要，可於 docs/06-pages.md `/guard` 段落補一句「過敏原區僅顯示疑似含有之類別」。
