# 運動紀錄：時長無預設、快捷分鐘與加入按鈕樣式

**日期**：2026-04-21  
**影響規格**：docs/06-pages.md  
**類型**：修改  

## 原規格

`/log` 運動分頁為手寫入 `activity_logs`，含類型與分鐘、估熱與備註（選填）；未細述時長預設值或快捷輸入。

## 實際做法

- `activity-log-section`：時長預設為空；提供 15／30／45／60／90 分鐘快捷按鈕帶入輸入框（可再修改）；有效分鐘才可送出；送出後清空時長。
- 「加入紀錄」按鈕為 Sea Green 綠底、白字、hover 加深綠。

## 原因

產品需求：避免預設時長誤填；常用時長一鍵帶入並可改；主要動作按鈕與品牌主色一致。

## 後續

可將 `docs/06-pages.md` 運動分頁列點補一句：時長可快捷選取或手填、無預設值。
