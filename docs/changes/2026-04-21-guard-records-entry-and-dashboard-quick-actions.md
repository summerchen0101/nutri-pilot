# 守衛標籤紀錄入口與首頁快速操作調整

**日期**：2026-04-21  
**影響規格**：docs/06-pages.md  
**類型**：修改

## 原規格

- `/guard` 頁首未提供「標籤紀錄」快捷入口。
- Dashboard 快速操作為 3 項（飲食、體重、數據）。

## 實際做法

1. `/guard` 頁首右上角新增「標籤紀錄」連結，導向 `/guard/records`。
2. 新增 `/guard/records` 頁面，查詢並顯示 `label_guard_saved_reports`（名稱、儲存時間、安全分數）；無資料時顯示空狀態與返回 `/guard` 入口。
3. Dashboard 快速操作調整為 5 項：飲食、體重、運動、數據、標籤（標籤連到 `/guard/records`）。

## 原因

使用者需要更快找到已儲存的標籤守衛結果，並在首頁直接進入運動與標籤相關紀錄流程。

## 後續

若後續增加守衛紀錄刪除/搜尋功能，可擴充 `/guard/records` 為可操作列表（分頁、關鍵字篩選）。
