# 飲食紀錄搜尋加入：份量改為滑桿

**日期**：2026-04-20  
**影響規格**：docs/03-features.md、docs/changes/2026-04-19-log-add-food-search-ui.md  
**類型**：修改  

## 原規格

搜尋加入流程含「快速份量按鈕」（依類別：一碗／一份／固定 ml 等）＋數字輸入與 g/ml 切換。

## 異動後行為

- 移除快速份量按鈕；改為單一水平 range 滑桿，預設區間 **100～1000**（與 g/ml 數字同一刻度；日後可由設定頁載入區間）。
- 保留數字輸入與 g/ml 切換；輸入失焦時將份量 clamp 至該區間。
- 寫入資料庫時份量與（未手動調整時的）巨量依 clamp 後數值計算，與區間一致。
