# 次級頁返回箭頭與快速操作樣式調整

**日期**：2026-04-21  
**影響規格**：docs/06-pages.md  
**類型**：修改

## 原規格

- 守衛頁右上角為文字「標籤紀錄」入口。
- 次級頁未統一左箭頭返回交互。
- Dashboard 快速操作包含外層白底卡片區塊。

## 實際做法

1. 守衛頁右上角改為「標籤紀錄 icon」入口，仍導向 `/guard/records`。
2. 次級頁（非底部導覽主頁）統一加入左箭頭返回，點擊使用 `router.back()`。
3. Dashboard 快速操作移除外層白底卡，改成分類按鈕列視覺（保留五項入口：飲食/體重/運動/數據/標籤）。

## 原因

統一次級流程的返回手勢與視覺語言，並降低首頁區塊層次，讓操作入口更貼近商城分類體驗。

## 後續

若後續新增更多次級頁，需沿用同一返回規則，避免頁面返回行為不一致。
