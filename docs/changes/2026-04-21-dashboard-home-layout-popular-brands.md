# Dashboard 首頁區塊與人氣品牌調整

**日期**：2026-04-21  
**影響規格**：docs/06-pages.md  
**類型**：修改

## 原規格

- Dashboard 主要描述了「今日餐食」與「快速操作列」兩個區塊，但未明確定義兩者先後順序。  
- 未定義 Dashboard 出現「本週人氣（品牌）」品牌卡列。

## 實際做法

- 將 Dashboard 的「快速操作」區塊上移到「今日餐食」之前。  
- 在「本週補給推薦」下方新增「本週人氣（品牌）」橫向卡列。  
- 先以 active 品牌隨機取最多 8 筆（且品牌需有上架商品）作為顯示資料。

## 原因

配合首頁資訊層級調整，讓常用操作更靠前，並在補給推薦後補上品牌探索入口，提升商城導流與逛感。

## 後續

後續可改為由後台維護「本週人氣品牌」排序與出現名單，取代目前隨機挑選邏輯。
